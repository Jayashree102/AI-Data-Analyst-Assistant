AI Data Analyst Assistant 🤖📊

Overview 📌
AI Data Analyst Assistant is an AI-powered data analysis website that allows users to upload datasets and interact with them using natural language queries. The system automatically generates insights, performs data cleaning, and provides statistical analysis along with interactive visualizations.

Features 🚀
- 🔍 Natural language querying on datasets
- 📊 Automated data insights generation
- 🧹 Data cleaning and preprocessing
- 📈 Statistical analysis
- 📉 Interactive data visualizations
- 🤖 AI-powered DataFrame agent for intelligent analysis

Tech Stack 🛠️
- Python 🐍
- Streamlit 🌐
- Pandas 📊
- Google Gemini API 🤖

Project Structure 📂
app.py  
core/agent.py  
utils/data_handler.py  
prompts/system_prompt.txt  
requirements.txt  
sample_sales_data.csv  

How to Run ▶️

1. Clone the repository:
git clone https://github.com/Jayashree102/AI-Data-Analyst-Assistant.git

2. Navigate to project folder:
cd AI-Data-Analyst-Assistant

3. Install dependencies:
pip install -r requirements.txt

4. Run the application:
streamlit run app.py

Example Use Cases 📊
- Sales data analysis
- Customer behavior insights
- Trend identification
- Data cleaning automation
